# Fiqh Feature

This package ships the Milestone 4 starter fiqh engine for the app.

Scope in this version:

- data-driven topic and ruling model
- local bundled starter knowledge pack
- checklist generation based on the selected fiqh profile
- disputed-issues browsing
- side-by-side school comparison support

Safety policy:

- the package presents sourced views, not fatwas
- each topic includes a scholar-escalation prompt for context-heavy cases
- the starter pack is intentionally small and conservative in scope

Current bundled topics:

- five daily prayers
- Ramadan fasting
- zakat al-mal
- Witr prayer
- shellfish and non-fish sea animals

Source policy:

- Quran references are stored as direct citations
- hadith references are stored as citation references
- school-specific differences use cited scholarly reference links
- the app should not silently rewrite or collapse differences into one ruling
