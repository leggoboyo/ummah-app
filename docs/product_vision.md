# Ummah App

Mission:
Create the most comprehensive Islamic mobile application available.

Core features:
- Accurate prayer times with Adhan notifications
- Qibla direction
- Quran reader with translations
- Hadith search
- Islamic calendar and Ramadan tools
- Fiqh knowledge engine
- AI assistant referencing Quran and hadith
- Support for Sunni and Shia jurisprudence profiles

Architecture:
Flutter mobile app
Offline-first design
Optional AI modules
